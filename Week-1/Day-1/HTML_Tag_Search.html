HTML Tag Search Assignment


b tag

The b tag in HTML is used to draw users attention to element's content , which are not otherwise granted special importance. This was known as boldface element. and the most browsers still draw the text in the boldface.

i tag

The i tag in the HTML represents a range of the text that is set off from the normal text for some reason, such as idiomatic text, technical terms, taxomonical designations, among others.

u tags

The u tags in the HTML represents a span of inline text which should be rendered in the way that indicates that it has a non-textual annonation. This is rendered by default as a simple solid underline, but may be altered using CSS

Select tag

The Select tag in HTML represents a control that provides a menu of options.

Button Tag

The select tag in HTML is an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology. Once activated, it then performs a programmable action, such as submitting a form or opening a dialog.
